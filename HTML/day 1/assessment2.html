HTML Cheatsheet

Quick Navigation

  • Anchor (<a>)
  • Image (<img>)
  • Paragraph (<p>)
  • Unordered List (<ul>)
  • Table (<table>)

<a> (Anchor Tag)

The <a> tag creates hyperlinks to other pages, files, locations, or URLs.

<a href="https://www.example.com" target="_blank">Visit Example!</a>

<img> (Image Tag)

The <img> tag is used to embed images; it requires the src attribute.

<img src="logo.png" alt="Site Logo" width="120" height="60">

<p> (Paragraph Tag)

The <p> tag defines paragraphs of text, adding spacing around content.

<p>This is a paragraph of text.</p>

<table> (Table Tag)

The <table> tag defines tabular data in rows and columns.

<table>
  <tr><th>Name</th><th>Score</th></tr>
  <tr><td>Alice</td><td>90</td></tr>
</table>
